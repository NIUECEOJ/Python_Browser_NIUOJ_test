# 建議事項

- 建議鎖定鍵盤之Win、Alt、Ctrl、Esc按鍵，可以使用PowerToys將鍵盤映射成Disable -> 已限制，已知問題無法禁用系統快速鍵，已使用高頻且禁用工作管理員方式解決

- 建議通知使用者關閉通知，否則跳出通知一樣會造成作弊

- 此程式需要搭配server接收logs(開發中)
